# 活動報告の追加方法

## 手順
1. `reports.json` を開く(編集ボタン=鉛筆アイコンをクリック)
2. 一番上(`[` の直後)に、下のテンプレートをコピペする
3. {{ }} の部分を実際の内容に書き換える
4. 直後に作った報告と、その下の報告の間に「,」(カンマ)があることを確認する
5. 画面下の「Commit changes」を押して保存する

## テンプレート(これをコピペして使う)

\`\`\`json
  {
    "date": "{{YYYY-MM-DD形式の日付。例: 2026-07-01}}",
    "title": "{{報告のタイトル}}",
    "text": "{{報告の内容。2〜3行程度で}}",
    "image": "{{使う画像のファイル名。例: houkoku0701.jpg}}"
  },
\`\`\`

## 画像のアップロード方法
1. `images` フォルダを開く
2. 「Add file」→「Upload files」
3. 写真をドラッグ&ドロップ
4. ファイル名を上のテンプレートの "image" の値と同じにする
5. 「Commit changes」

## 注意点
- 日付の形式は必ず「2026-07-01」のように西暦-月-日
- 文字列は必ず " " (ダブルクオート)で囲む
- 一番最後のブロックの後ろにはカンマをつけない(エラーになる)
